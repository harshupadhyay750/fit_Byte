# FitByte – AI-Powered Diet Planner & Nutrition Management App

FitByte is a premium fitness application built with Flutter, Firebase, and OpenAI API. It helps users track nutrition, calories, water intake, and weight progress through a modern Material 3 interface.

## 🚀 Features

- **Splash & Onboarding**: Animated splash screen and modern onboarding flow.
- **Authentication**: Secure login/register with Firebase.
- **AI Diet Planner**: Personalized meal plans generated by AI.
- **Calorie Tracker**: Log meals and track daily intake.
- **Macro Analytics**: Detailed charts for protein, carbs, and fats.
- **Water Tracker**: Monitor daily hydration goals.
- **Weight Tracker**: Visualize progress over time.
- **AI Chatbot**: Instant nutrition advice and meal suggestions.

## 🛠 Tech Stack

- **Framework**: Flutter
- **Language**: Dart
- **Backend**: Firebase (Auth, Firestore, Storage)
- **State Management**: Riverpod
- **Design**: Material 3
- **Charts**: fl_chart
- **Animations**: flutter_animate & Lottie

## ⚙️ Setup Instructions

1. **Clone the repo**
2. **Install dependencies**: `flutter pub get`
3. **Firebase Setup**:
   - Create a project on [Firebase Console](https://console.firebase.google.com/).
   - Add Android/iOS apps.
   - Run `flutterfire configure` to generate `firebase_options.dart`.
4. **OpenAI API**:
   - Get an API key from [OpenAI](https://platform.openai.com/).
   - Add it to `lib/services/ai_service.dart`.
5. **Run the app**: `flutter run`

## 🏗 Build Instructions

- **Android APK**: `flutter build apk --release`
- **iOS App Store**: `flutter build ipa`
